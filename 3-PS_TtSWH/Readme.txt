 The reason I created this program.

 This is an assignment on Coursera. For training Graph, LinkedList, Dictionaries
It is a good opportunity to practice Dijkstra's algorithm. I considered it very hard 
to understand both algorithms and how to implement them with new knowledge.

Courseraの課題です。 グラフ、LinkedList、Dictionariesの練習ため
Dijkstra's algorithmを練習する良い機会です。 アルゴリズムと、
新しい知識で実装する方法を理解するのは非常に難しいと思いました。

-----------------------------------------------------------------------------
 What do I find difficult
 プログラムを作成する上で大変だった所
-----------------------------------------------------------------------------
 This is the hardest assignment across all courses I take on Coursera, in my opinion.
Start by taking each waypoint information in GraphBuilder class. 
I need to add weight to each node for calculating later in this assignment
(If I miscalculate, it will be harder to debug later). The next thing is Reposition method 
in SortedLinkedList class. My code is much longer than this before improving optimization. 
And the most difficult is the Search method in the Traveler class. 
I spend a really long time identifying algorithms and implementing Graph, LinkedList, Dictionary.
And much more time to debug this chunk of code.

私の意見では、このCourseraですべての受講したコースの中で最も難しい課題と思います。
GraphBuilderクラスの各ウェイポイント情報を取得することから始めます。
この課題の後半で計算するために、各ノードに重みを追加する必要があります
（計算を誤ると、後でデバッグするのが難しくなります）。
次は、SortedLinkedListクラスのReposition medthodです。
私のコードは、最適化される前はこれよりとても長いです。
そして、最も難しいのは、TravelerクラスのSearch methodです。
長い時間をかけてアルゴリズムを特定し、Graph、LinkedList、Dictionaryを実装しています。
そして、このコードの部分をデバッグするときに多くの時間を使っていました。
